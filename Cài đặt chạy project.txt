Cài đặt chạy project
Cài đặt vite 

npm install --save-dev vite laravel-vite-plugin

Cài đặt vue cho project

npm install vue

Cài đặt axios lấy API

npm install axios

Cài đặt plugin vue cho vite

npm install @vitejs/plugin-vue

//Cài đặt vuex

//npm install vuedraggable@next --save

Cài đặt kéo thả  vue-draggable-next

npm install vuedraggable@next

Cài đặt chạy một lệnh duy nhất cho project

npm install npm-run-all --save-dev

Cài đặt boostrap

npm install bootstrap @popperjs/core

Sau đó chạy lệnh 

npm run dev
